# Task 5 - Final PHP CRUD Project

This is the final project for ApexPlanet Internship.

## Features
- User Registration and Login
- CRUD Operations (Add, View, Delete Posts)
- Search Posts (Title / Content)
- Pagination (5 posts per page)
- Bootstrap UI Improvements
- Security Enhancements:
  - Prepared Statements
  - Form Validation
  - Role Based Access (Admin/User)

## Admin and User Roles
- Admin can delete posts
- User cannot delete posts

## How to Run the Project
1. Start Apache and MySQL in XAMPP
2. Open phpMyAdmin and create database `blog`
3. Create tables `users` and `posts`
4. Run this project in browser:

http://localhost/task5-final/login.php
